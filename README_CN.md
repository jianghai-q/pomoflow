# 🍅 PomoFlow - 给电脑小白的使用指南

## 🌟 最简单的方式：在线使用

> 推荐！无需安装任何软件，直接在浏览器中访问

**网址：** `https://pomoflow.vercel.app` （部署后更新）

---

## 💻 本地运行

### 第一步：安装 Node.js（5分钟）

1. 访问 https://nodejs.org/zh-cn/
2. 点击下载 **LTS 版本**（推荐版本）
3. 双击下载的安装包
4. 一路点击"下一步"完成安装

### 第二步：运行项目（1分钟）

**双击这个文件：** `启动项目.bat`

等待片刻，浏览器会自动打开应用！

---

## 📋 手动运行（如果自动启动失败）

1. 找到 `pomoflow` 文件夹
2. 按住 `Shift`，右键点击空白处
3. 选择"在此处打开命令窗口"
4. 首次运行输入：`npm install` 回车
5. 然后输入：`npm run dev` 回车
6. 在浏览器打开：http://localhost:5173

---

## ❓ 常见问题

| 问题 | 解决方法 |
|------|----------|
| 双击 bat 没反应 | 右键 → 以管理员身份运行 |
| 提示 npm 命令不存在 | 需要先安装 Node.js |
| 浏览器没自动打开 | 手动访问 http://localhost:5173 |
| 如何停止程序 | 关闭命令窗口即可 |
