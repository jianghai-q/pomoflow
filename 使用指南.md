## 💻 方法二：本地运行

### 给开发者的部署步骤

**方式A：Vercel 部署（5分钟搞定）**

1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New" → "Project"
4. 点击 "Continue with GitHub"
5. 选择 `jianghai-q/pomoflow` 仓库
6. 点击 "Deploy"，等待部署完成
7. 部署成功后，复制访问网址

**方式B：Netlify 部署**

1. 访问 https://app.netlify.com/drop
2. 将 `pomoflow` 文件夹中的 `dist` 文件夹拖拽到网页上
3. 等待上传完成，获得网址

---

### 给使用者的操作步骤

#### 📋 前置要求：安装 Node.js

**如何检查是否已安装？**
1. 按 `Win + R` 打开运行窗口
2. 输入 `cmd` 回车
3. 输入 `node -v` 回车
4. 如果显示版本号（如 v18.x.x），说明已安装

**如果未安装：**
1. 访问 https://nodejs.org/zh-cn/
2. 下载 "LTS" 版本（推荐版本）
3. 双击下载的安装包
4. 一路点击"下一步"完成安装

#### 🖱️ 运行项目

**方式A：双击启动脚本（推荐）**

1. 找到 `启动项目.bat` 文件
2. 双击运行
3. 等待自动启动

**方式B：手动启动**

1. 找到 `pomoflow` 文件夹
2. 按住 `Shift` 键，右键点击空白处
3. 选择"在此处打开 PowerShell 窗口"或"在此处打开命令窗口"
4. 输入 `npm install` 回车（仅首次运行需要）
5. 输入 `npm run dev` 回车
6. 等待显示 "Local: http://localhost:5173/"
7. 在浏览器中打开 http://localhost:5173

#### 📌 常见问题

**Q: 双击 bat 文件没有反应？**
A: 右键点击 → "以管理员身份运行"

**Q: 显示 "npm 不是内部或外部命令"？**
A: 需要先安装 Node.js（见上方前置要求）

**Q: 浏览器没有自动打开？**
A: 手动在浏览器地址栏输入 http://localhost:5173 并访问

**Q: 如何停止程序？**
A: 关闭命令窗口即可

**Q: 更新项目后需要重新安装依赖吗？**
A: 只需运行 `npm install` 一次，后续更新直接运行 `npm run dev`
